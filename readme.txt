
Our group have chosen the program Peopledetect.cpp to work on.
We change the original data to the ones we found on the internet. We have downloaded some files such as test1.mp4, NYC.jpg, humans1.jpg, humans2.jpg, cricket.jpg and football1.jpeg.
*******************************************************
HOW TO USE:
Video:  Enter the path of the video file you want to read, in between the | | in Line 19.
Image:  Enter the path of the image file you want to read, in between the | | in Line 17.
Camera: To use camera, type './peopledetect -c -1', without the quotes, when you run the program.
By default, the program will run on one of our own video sample test1.mp4
